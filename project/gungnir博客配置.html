# 安装

详情访问官网：https://v2-vuepress-theme-gungnir.vercel.app/

# 步骤

  1. 创建项目

    mkdir blog && cd blog
npm init

  2. 安装 VuePress 和主题 Gungnir：

    npm install -D vuepress@2.0.0-beta.49 vuepress-theme-gungnir@next

    49版本以后去除了theme API layouts，不要使用

  3. 添加npm脚本

    {
  "scripts": {
    "docs:dev": "vuepress dev docs --clean-cache",
    "docs:build": "vuepress build docs --clean-cache"
  }
}

  4. 在【根目录/docs/.vuepress/config.js中进行配置】

    // .vuepress/config.js

const { gungnirTheme } = require("vuepress-theme-gungnir");

module.exports = {
  ...
  theme: gungnirTheme({
    // 主题具体配置
  })
}

# front matter

用于设置每个md文件怎么被转成页面

# 常用配置

  • layout：用于告诉vuepress把这个md文件放到什么模板的页面中。具体有什么可以选择的，到vuepress-theme-gungnir——lib——client——layouts中查看

    • HomePage：首页
    • Post：普通博客页
    • Links：外部链接页

  • home：表示这个是首页。点击顶栏左上角会到这里

  • title：标题，同时会出现在浏览器的标签中

  • subtitle：副标题

  • date：yyyy-MM-dd，博客发表日期

  • author：作者，不填的话默认是config中的personalInfo.name属性

  • useHeaderImage：是否在当前博客中显示封面

  • headerImage：当前博客的封面图

  • catalog：是否禁用侧栏，默认true

  • giscus：是否禁用评论，默认true

  • hide：是否在首页和标签页列表中隐藏当前博客

  • tags：添加方式，在下面加上【-】然后空一格，写上tag名

    tags:
	- a
	- b

# 首页配置例子

layout: HomePage
home: true
title: 首页
---

# 常用标签

h1~6

img

a

table、tr、td、th

ul、ol、li

dl、dt、dd：dl包括dt和dd，dt和dd是兄弟，若干个dd描述一个dt

form、input

label：一个label标签绑定一个表单标签，点击label的内容时，就会把焦点放到对应的表单标签，方便扩大点击范围

select、option

checkbox

textarea

# 常用属性

a标签的href属性

td标签的rowspan、colspan属性

input标签的type、name、value、checked、maxlength属性

label标签的for属性：值是某个表单标签的id，可以实现绑定

textarea的rows、cols标签

# H5新特性

兼容性有问题，IE9+

以前全是div，H5有语义化标签：可读性和维护性更强。header、nav、article、aside、section、footer等，但是不能兼容IE8

audio、video：autoplay自动播放、controls显示播放控件、loop是否自动循环、poster加载等待图片、muted静音、preload是否预加载

input的新type：date、color、number、email、url、time、month、week、tel、search

input的新属性：required必填、placeholder提示信息、autofocus自动聚焦、autocomplete让你选择之前输入过的选项（默认打开）、multiple多选文件

# HTML骨架

!DOCTYPE html 文档类型说明

html lang="en"对页面主要语言进行声明

meta charset="UTF-8" meta标签用来描述网页文档的属性，此处为使用utf-8的编码格式

meta http-equiv="X-UA-Compatible" content="IE=edge"使用IE浏览器当前最高版本的内核来渲染，严格模式

meta name="viewport" content="width=device-width, initial-scale=1.0"视口设置
